Install Composer 2.9.0

## Description

This patch installs Composer version 2.9.0 on your deployment. Composer 2.9.0 includes performance and security improvements. You can read more about the release [here](https://blog.packagist.com/composer-2-9/).

## Impact

There will be downtime as new AMI images are rolled out to your deployment.

## Changes

- Install composer 2.9.0 on jump and web nodes
- Makes 2.9.0 available as a selectable composer version in your deployment

## If You Accept

Composer 2.9.0 will be installed on your deployment and available to select. Your current composer version will not be changed automatically.

## If You Reject

Composer 2.9.0 will not be available until you upgrade your deployment or accept this patch.
